GitHub Pages upload instructions

1. Upload every file inside this folder to the root of the GitHub Pages repository.
2. index.html must be at the top level of the repository, not inside another folder.
3. The live app link should end with the repository name, /, or /index.html.
4. If the page only shows an app icon, the browser is probably opening /icons/icon-192.svg or /icons/icon-512.svg instead of /index.html. Change the address to the root app URL and delete/re-add the iPhone Home Screen shortcut if needed.
5. Service worker caching is disabled in this version to prevent stale blank pages.
